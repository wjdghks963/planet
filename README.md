# planet

식물을 일기로 관리하고 다른 사용자들의 식물과 일기를 보며 다른 식물에 대한 정보를 얻을 수 있는 소셜 앱입니다.

AI를 이용해 유저들의 식물을 분석하고 궁금한 점을 물어볼 수 있습니다.

## libraries

flutter, getx, http etc. 


## Design pattern MVVM

getx 사용해 mvvm 패턴 적용

getx의 controller 사용함
